传统 JNI 开发步骤

总述

Android 的 JNI 开发一般可以总结为以下步骤

  1. 编写含有 Native 方法的 Java 类
  2. 用 javah 生成 C/C++ 原生函数的头文件
  3. 用 C/C++ 实现原生函数
  4. 将项目依赖的原生库和资源添加到工程中
  5. 生成应用
