Intellj IDEA 的使用

2015-09-11 - 阅读了大约 3 分钟

快捷键

快捷键 说明
cmd+n 查找类
control+n 生成代码
cmd+alt+l 格式化
cmd+f 查找文本
cmd+r 查找替换
cmd+p 方法参数提示
control+j quick doc
cmd+shift+n 查找文件
cmd+alt+shift+n 查找类中的成员
shift+f6 重命名
cmd+y 删除行
cmd+o 重载父类方法
cmd+e 最近访问列表
cmd+b 跳转到声明
cmd+] 跳转到括号的结尾
alt+shift+c 对比最近代码
cmd+w 选择词，行，代码块
control+h 查看类的继承层次
cmd+alt+t surround with
ctrl+shift+t 建立测试类
或者在类与测试类中跳转
shift+cmd+delete 上次编辑位置
cmd+alt+m 提取代码块为方法
cmd+alt+p 提取变量为参数
cmd+alt+v 提取为局部变量
control+option+h 显示方法树
fn+option+f7 显示变量的调用处
cmd+shift+a 显示action 窗口
option+enter 创建Intention Actions
